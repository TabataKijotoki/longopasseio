# longopasseio

Design system e protótipo mobile para o app Longo Passeio — curadoria de cafés no centro de São Paulo.

## Estrutura

- `project/` — design system completo (tokens, preview cards, UI kit)
- `project/ui_kits/app/` — protótipo React/Babel rodável (`index.html`)
- `project/colors_and_type.css` — tokens de cor e tipografia
- `chats/` — transcrições do design session
- `backend/` — servidor de autenticação
